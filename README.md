# Signup Form with Dashboard (VanillaJS)

A simple signup system built using HTML, CSS, and JavaScript. Users can enter their details through a form and the data is dynamically displayed in a dashboard table.

## Features

* User signup form with input validation and hidden password(s)
* Dynamic table to display user data
* Ability to delete users from the table
* User-friendly interface

## Tech Stack

* HTML
* CSS
* Vanilla JS
